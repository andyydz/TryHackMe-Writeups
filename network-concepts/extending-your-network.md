# Task 1 — Introduction to Port Forwarding

In this task, I learned what port forwarding actually is.

So basically, when someone from the internet tries to access something inside your local network, the router needs to know where to send that traffic. That is where port forwarding comes in.

For example:

- Someone enters your public IP
- Router forwards that traffic to a specific device inside the network

### Things I understood

- Difference between public IP and private IP
- Basic idea of NAT
- Why ports are important
- How routers handle traffic

### Real-world uses

- Hosting game servers
- Websites
- Remote access
- CCTV cameras

# Task 2 — Firewalls 101

This section was about firewalls.

So basically, a firewall acts like a filter or security guard for network traffic. It checks traffic coming in and going out and then decides whether to allow it or block it depending on rules.

I also learned about two types of firewalls.

### Stateless Firewall
Checks packets individually without remembering previous traffic.

### Stateful Firewall
Keeps track of active connections and makes smarter decisions.

### Main thing I learned

Firewalls are super important because they help stop unauthorized access.

# Task 4 — VPN Basics

This task explained VPN basics.

A VPN basically creates an encrypted tunnel between your device and another network/server. It helps protect privacy and makes internet usage more secure.

Especially useful on public Wi-Fi.

### Benefits of VPN

- Encrypts traffic
- Better privacy
- Hides IP address
- Safer browsing

### VPN protocols mentioned

- OpenVPN
- WireGuard
- IPSec

# Task 5 — LAN Networking Devices

This section explained some common networking devices.

### Router
Connects different networks together.

### Switch
Connects devices inside the same network and sends traffic efficiently.

### Hub
Sends traffic to every connected device.

### Access Point
Provides Wi-Fi connectivity.

This task made the differences between these devices much easier to understand.

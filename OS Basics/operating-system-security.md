# Operating System Security - TryHackMe Write-up

## Overview
This room introduced the fundamentals of Operating System (OS) Security and explained how operating systems implement security mechanisms to protect systems, users, files, and services from unauthorized access and cyber threats.

The module also included practical exercises within a virtual machine environment to demonstrate real-world examples of operating system security concepts and defensive measures used in enterprise environments.

Understanding OS security is essential in cybersecurity because operating systems act as the foundation for user access, applications, system processes, and enterprise infrastructure.

# Access Services & Security Controls

Operating systems manage and control access to system resources, files, users, and services.

## Access Services Involved
- User authentication
- File and directory permissions
- Access control mechanisms
- User account management
- Process and service management
- System-level security protections

These services help ensure that only authorized users and applications can access sensitive resources.

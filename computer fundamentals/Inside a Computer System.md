# Inside a Computer System | TryHackMe Write-up

In this room, I learned about the basic components inside a computer system and how they work together when a computer starts up and runs programs. Before this, I knew the names of some hardware components, but I did not fully understand their actual purpose or how they interact with each other internally.

This room helped me understand the foundation of how computers operate, which is useful not only for general IT knowledge but also for cybersecurity and troubleshooting.

---

# Main Components Inside a Computer

One of the main things I learned is that a computer is made up of multiple hardware components, and each component has a specific role.

All these parts work together to process data, store information, and run applications.

---

# CPU (Central Processing Unit)

The CPU is basically the brain of the computer.

It processes instructions and performs calculations required by programs and the operating system.

Whenever we:

- Open applications
- Browse websites
- Run programs
- Play games

the CPU is constantly processing instructions in the background.

### Real-world example

If someone opens multiple applications at once, the CPU handles all those tasks and decides how system resources are used.

### What I understood

A faster CPU usually means better performance because instructions can be processed more quickly.

---

# RAM (Random Access Memory)

RAM is temporary memory used by the computer while programs are running.

Unlike storage devices, RAM only stores data temporarily while the system is powered on.

### Example

When opening a browser or editing software, the application gets loaded into RAM so it can be accessed quickly.

If the system has low RAM:

- Applications may become slow
- Multitasking becomes difficult
- System lag may happen

### Real-world importance

This helped me understand why systems with more RAM usually perform better during multitasking.

---

# Storage Devices

This section explained storage components like HDDs and SSDs.

Storage devices permanently save:

- Operating systems
- Applications
- Files
- Games
- Documents

### HDD (Hard Disk Drive)

Uses spinning disks to store data.

### SSD (Solid State Drive)

Uses flash memory and is much faster than HDDs.

### What I understood

SSDs improve:

- Boot speed
- File transfer speed
- Overall system responsiveness

This is why modern systems mostly use SSDs now.

---

# Motherboard

The motherboard connects all computer components together.

It allows communication between:

- CPU
- RAM
- Storage devices
- GPU
- Other hardware

Without the motherboard, components would not be able to interact properly.

### Real-world example

Whenever data moves from storage to RAM or CPU, the motherboard helps coordinate that communication.

---

# Power Supply Unit (PSU)

The PSU provides electrical power to all components inside the computer.

Different components require different amounts of power, and the PSU distributes power accordingly.

### What I learned

Without a stable power supply, the system may:

- Crash
- Restart unexpectedly
- Fail to boot properly

---

# GPU (Graphics Processing Unit)

The GPU handles graphics-related tasks.

It is responsible for rendering:

- Videos
- Games
- Animations
- Visual effects

### Real-world use

Gaming, video editing, AI processing, and 3D rendering rely heavily on GPUs.

### What I understood

A dedicated GPU can greatly improve graphical performance compared to integrated graphics.

---

# What Happens When You Press the Start Button?

This was one of the most interesting parts of the room because it explained the boot process.

When the power button is pressed:

1. PSU supplies power to components
2. BIOS/UEFI starts running
3. Hardware checks happen (POST)
4. Bootloader is located
5. Operating system starts loading into RAM
6. User login screen appears

Before this room, I never really thought about how many processes happen within seconds during startup.

---

# BIOS and UEFI

I also learned about BIOS/UEFI.

These are firmware systems that initialize hardware before the operating system starts.

### Their role

- Check hardware
- Start the boot process
- Load the operating system

### Real-world importance

If BIOS settings are incorrect or corrupted, the system may fail to boot properly.

---

# Why This Knowledge Matters

This room made me realize that understanding computer hardware is important for many areas:

- Troubleshooting
- System building
- Performance optimization
- Cybersecurity
- Digital forensics

For cybersecurity especially, understanding hardware and system behavior helps when analyzing attacks, malware behavior, or system performance issues.

---

# Overall Understanding

Overall, this room gave me a much clearer understanding of how computer systems work internally.

Before this, I mostly used computers without thinking about what happens inside the machine, but now I better understand:

- The role of each hardware component
- How the boot process works
- Why RAM and CPUs affect performance
- How hardware components communicate

This room was beginner-friendly and useful for building a strong foundation in computer fundamentals.

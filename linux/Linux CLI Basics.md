# Linux CLI Basics - TryHackMe Write-up

## Overview
This room introduced the fundamentals of the Linux Command Line Interface (CLI) and provided hands-on experience working in a Linux environment through a virtual machine. The module focused on system navigation, file management, basic Linux commands, and understanding how Linux systems are commonly used in cybersecurity, SOC operations, and enterprise environments.

Linux is widely used in cybersecurity because many servers, cloud systems, security tools, and enterprise infrastructures rely heavily on Linux-based operating systems.

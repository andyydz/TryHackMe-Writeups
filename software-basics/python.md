# Python Concepts Explored

## Variables
Variables are used to store information and values inside programs.

### Example

```python
name = "Andrew"
age = 20
```

Variables help programs manage and process data efficiently.

---

## Conditional Statements
Conditional statements allow programs to make decisions based on conditions.

### Example

```python
if age >= 18:
    print("Adult")
else:
    print("Minor")
```

Conditions are important in automation and logical processing.

---

## Iterations & Loops
Loops are used to repeat tasks automatically.

### Example

```python
for i in range(5):
    print(i)
```

Iterations help automate repetitive tasks efficiently.

# JavaScript Concepts Explored

## Variables
Variables are used to store information and values inside programs.

### Example

```javascript
let username = "Andrew";
let age = 20;
```

Variables help applications manage user and system data dynamically.

---

## User Input
JavaScript can collect user input through prompts and forms.

### Example

```javascript
let name = prompt("Enter your name");
```

This allows web applications to interact with users dynamically.

---

## Conditional Statements
Conditional statements help programs make decisions based on user input or conditions.

### Example

```javascript
if (age >= 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}
```

Conditions are important for authentication systems, validations, and application logic.

---

## Iterations & Loops
Loops allow repetitive tasks to be automated.

### Example

```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```

Iterations improve efficiency and automation within programs.

# Data Encoding Concepts

Data encoding allows computers to convert human-readable text into machine-readable numerical values.

## Encoding Standards Explored

### ASCII
ASCII (American Standard Code for Information Interchange) is a character encoding standard used to represent English letters, numbers, and symbols.

### Example
```text
A = 65
B = 66
a = 97
```

ASCII is commonly used in:
- Text files
- System logs
- Command-line environments
- Basic data processing

---

### Unicode
Unicode extends character support beyond ASCII and allows systems to represent multiple languages, symbols, and international characters.

### Example
Unicode supports:
- Emojis
- International languages
- Special symbols
- Multilingual applications

Unicode is widely used in:
- Modern software applications
- Websites
- Databases
- Communication systems

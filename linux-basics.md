# Linux Basics - TryHackMe

## What this room was about
This room introduced basic Linux commands and how to navigate and manage files using the terminal.

## What I learned
- Linux is a command-line based operating system
- Navigation commands like `pwd`, `ls`, and `cd`
- File operations like `touch`, `mkdir`, `rm`
- Viewing file contents using `cat`
- Importance of permissions and file structure

##  Commands I practiced
- `pwd` → shows current directory
- `ls` → lists files and folders
- `cd` → change directory
- `mkdir` → create new directory
- `touch` → create a file
- `rm` → remove file
- `cat` → view file contents

##  Key Takeaway
Understanding basic Linux commands is essential for working in cybersecurity, as most tools and systems use Linux.

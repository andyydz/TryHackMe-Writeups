📄 **Full Writeup:** [digital-forensics-fundamentals.md](./digital-forensics-fundamentals.md)

## Skills Covered

| Skill | Tool / Concept |
|---|---|
| Forensic Methodology | Collection → Examination → Analysis → Reporting |
| Evidence Integrity | Chain of custody, write blockers, proper authorization |
| Disk & Memory Forensics | FTK Imager, Autopsy, Volatility |
| Metadata Analysis | `pdfinfo`, `exiftool` (EXIF/GPS extraction) |
| Forensic Specialties | Network, Disk, Memory, Mobile, Malware, Email, Cloud, Database forensics |

## Key Takeaway

Every forensic discipline reduces to the same four-stage methodology applied to a different evidence type — and metadata (PDF info, EXIF data) is often the fastest path to cracking a case.

# Cryptographic Concepts Explored

## Hiding Information
Cryptography helps protect sensitive data from unauthorized access by converting readable information into unreadable formats.

### Example
Plain text:
```text
HELLO
```

Encrypted text:
```text
KHOOR
```

This process helps secure information during transmission and storage.

---

## Symmetric Encryption
Symmetric encryption uses the same key for both encryption and decryption.

### Example
- One shared secret key
- Faster encryption process
- Commonly used for bulk data encryption

### Real-world Usage
- File encryption
- VPN communication
- Secure storage systems

---

## Sharing Keys Safely
One challenge in symmetric encryption is securely sharing the encryption key between users.

If the key is intercepted, attackers may gain access to encrypted information.

This led to the development of asymmetric encryption methods.

---

## Asymmetric Encryption
Asymmetric encryption uses two keys:
- Public Key
- Private Key

### Example
- Public key encrypts the data
- Private key decrypts the data

### Real-world Usage
- HTTPS websites
- Secure email communication
- Digital certificates
- Authentication systems

Asymmetric encryption improves security by eliminating the need to share private keys publicly.

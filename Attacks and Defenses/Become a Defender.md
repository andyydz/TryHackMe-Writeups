# Defensive Security Concepts Explored

## What is Defensive Security
Defensive security focuses on protecting systems, detecting threats, and responding to cyber incidents.

The primary goals include:
- Preventing attacks
- Detecting suspicious activity
- Monitoring environments
- Responding to security incidents
- Protecting enterprise infrastructure

---

## Understanding Your Environment
Defenders must understand how systems, networks, users, and services interact within an environment.

### Example
A defender may monitor:
- User activity
- Network traffic
- Login attempts
- Endpoint behavior
- Security alerts

Understanding normal system behavior helps analysts detect anomalies and threats more effectively.

---

## Defending Your Environment
Organizations use multiple security controls and monitoring solutions to defend systems from attacks.

### Defensive Measures
- Firewalls
- Antivirus solutions
- Security monitoring tools
- Access controls
- Intrusion detection systems
- Security policies

These controls help reduce risks and improve enterprise security posture.

# Threat Investigation Applications

Defensive security knowledge supports incident response and threat investigations.

## Threat Investigation Tasks
Security professionals may:
- Analyze security logs
- Investigate alerts
- Identify malicious activity
- Track attacker behavior
- Review endpoint activity
- Contain security incidents

### Example
Indicators of suspicious activity may include:
- Multiple failed logins
- Unusual network connections
- Unexpected file changes
- Suspicious processes
- Unauthorized privilege escalation

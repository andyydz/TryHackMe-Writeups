# CMD Commands

## Overview
CMD (Command Prompt) is a Windows command-line interpreter used to execute commands, troubleshoot systems, manage files, and perform administrative tasks.

---

# Navigation Commands

## dir
Lists files and folders.

```cmd
dir
```

---

## cd
Changes directory.

```cmd
cd Desktop
cd ..
```

---

# File Management Commands

## mkdir
Creates a folder.

```cmd
mkdir test
```

---

## del
Deletes files.

```cmd
del file.txt
```

---

## copy
Copies files.

```cmd
copy file.txt backup.txt
```

---

## move
Moves files.

```cmd
move test.txt Documents
```

---

# Networking Commands

## ipconfig
Displays IP configuration.

```cmd
ipconfig
ipconfig /all
```

---

## ping
Checks connectivity.

```cmd
ping google.com
```

---

## tracert
Traces route to destination.

```cmd
tracert google.com
```

---

# System Commands

## tasklist
Displays running processes.

```cmd
tasklist
```

---

## systeminfo
Displays system information.

```cmd
systeminfo
```

---

## whoami
Displays current user.

```cmd
whoami
```

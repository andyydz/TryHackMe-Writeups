# Windows CLI Basics - TryHackMe Write-up

## Overview
This room introduced the fundamentals of the Windows Command Line Interface (CLI) and provided hands-on experience interacting with Windows systems through terminal-based commands. The module focused on file navigation, system information gathering, and understanding how command-line tools are used in cybersecurity, SOC operations, and system administration.

The room also included practical exercises within a virtual machine environment to help build familiarity with Windows command-line operations used in real-world environments.

# Active Directory Concepts Explored

## Windows Domain
A Windows Domain is a centralized environment used to manage users, computers, permissions, and resources across an organization.

### Benefits of Domains
- Centralized administration
- User authentication management
- Policy enforcement
- Enterprise resource management

Domains help organizations manage large environments efficiently.

---

## Active Directory (AD)
Active Directory is Microsoft's directory service used to manage:
- Users
- Computers
- Permissions
- Authentication
- Security policies

AD acts as the central management system in many enterprise environments.

---

## Managing Users in AD
User management involves:
- Creating user accounts
- Managing permissions
- Resetting passwords
- Organizing users into groups

### Security Relevance
Proper user management helps:
- Reduce unauthorized access
- Enforce least privilege
- Improve account security
- Monitor enterprise users

---

## Managing Computers in AD
Computer management involves organizing and controlling enterprise devices connected to the domain.

### Administrative Tasks
- Joining systems to domains
- Applying security policies
- Managing organizational units (OUs)
- Monitoring enterprise endpoints

This helps organizations maintain centralized control over systems.

---

## Group Policies
Group Policies allow administrators to apply security and configuration settings across systems and users.

### Example Policies
- Password policies
- Desktop restrictions
- Security configurations
- Software restrictions

Group Policies help standardize and secure enterprise environments.

---

## Authentication Methods
Authentication methods verify the identity of users and systems before granting access.

### Examples
- Username & password authentication
- Domain authentication
- Kerberos authentication

Authentication systems help protect enterprise resources from unauthorized access.

---

## Trees, Forests & Trusts

### Trees
A tree represents multiple connected domains sharing a common structure.

### Forests
A forest is a collection of multiple domain trees connected together.

### Trusts
Trust relationships allow domains or forests to share resources securely.

These structures help organizations scale and manage large enterprise environments.

Active Directory knowledge is highly important in cybersecurity because AD environments are frequently targeted during cyberattacks.

## SOC Analyst Applications
AD knowledge helps SOC Analysts:

- Detect privilege escalation attempts
- Investigate compromised accounts
- Review Group Policy changes

### Example
A SOC Analyst may investigate:
- Suspicious domain logins
- Unauthorized account creation
- Privilege escalation attempts

- Lateral movement within a domain
